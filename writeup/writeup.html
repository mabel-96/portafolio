Mitigación

Proporciona recomendaciones sobre cómo mitigar o corregir la vulnerabilidad explotada.

Recomendaciones de Seguridad

  • Utilizar consultas parametrizadas para prevenir SQLi
  • Actualizar el software vulnerable a la última versión
  • Implementar un WAF para proteger contra ataques web comunes
  • Seguir el principio de mínimo privilegio

Código corregido como ejemplo: 

// Código corregido
                    $id = $_GET['id'];
                    $stmt = $conn->prepare("SELECT * FROM products WHERE product_id = ?");
                    $stmt->bind_param("i", $id); // Consulta parametrizada segura
                    $stmt->execute();
                    $result = $stmt->get_result();
